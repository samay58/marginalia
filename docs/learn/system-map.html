4
Edit
Type, diff, rail updates

Each keystroke flows through a debounced reactive chain. The editor stays responsive while expensive diffing batches in the background.

Editor.svelte markdownUpdated listener
-> onPlainTextChange(text)
app.js updatePlainText(text)
-> editedPlainText.set(text) [immediate]
-> setTimeout(120ms)
  -> debouncedEditedPlainText.set(text)
    -> diffResult recomputes
      -> substantiveChanges / trivialChanges
ChangeRail.svelte re-renders

Token-level diffing. Change IDs reconciled for annotation stability.

User sees: green/red highlights in text. Change rail populates.

5
Annotate
Rationale saved

Click a change, write a rationale. The annotation stores target metadata so it can survive future edits. Resolution runs on every diff update.

+page.svelte selectChange() -> handleStartCompose()
annotations.js createAnnotationRecord()
-> buildAnnotationTarget(change, editedText)
  Stores: changeId, excerpt, line, context, blockKey
app.js resolvedAnnotations (derived)
-> resolveAnnotations(): exact match / reattach / stale

Reattachment scoring: 60% excerpt, 18% line, 8% before, 8% after, +blockKey bonus, -distance penalty. Min 0.74, margin 0.12.

User sees: note badge in gutter. Annotation in right column.

6
Finalize
Bundle to disk

Press Escape or Cmd+Enter. The app computes a fresh diff, generates all 7 bundle files, saves them via Rust, writes a status JSON, closes.

+page.svelte handleDone()
-> computeDiff() (fresh, not debounced)
-> computeSemanticChanges()
bundle.js generateBundle()
-> 7 files: original.md, final.md, changes.json,
  annotations.json, changes.patch,
  provenance.json, summary_for_agent.md
Rust save_bundle() -> write_file(status)

Bundle goes to ~/.marginalia/bundles/[timestamp]_[filename]/

User sees: window closes.

Handoff

hooks/post-write.sh

output_hook_response("Read feedback at:

summary_for_agent.md: changes count, feedback by priority, general notes.

User sees: they are back in their terminal. Agent is working.
